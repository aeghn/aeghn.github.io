保存网页

07 Feb 2021 / linux

最近发现了一个用来保存网页的命令行小工具 monolith, 很好用。

几行脚本，保存网页到指定目录。 

#!/usr/bin/env bash

WEB_PAGE_DIR=$HOME/Documents/reader_webpage

URL="$1"

mkdir -p "$WEB_PAGE_DIR"
cd "$WEB_PAGE_DIR"

proxychains -q monolith "$URL" -o $(echo "$URL" | sed 's/[^[:alpha:]]/_/g').html
